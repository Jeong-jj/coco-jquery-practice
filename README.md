# Design coco website

## 개요

- 주제: 디자인 컴퍼니 홈페이지
- 언어: HTML / CSS / jQuery
- URL: https://jeong-jj.github.io/coco-jquery-practice/

<br/>

### 구현사항

- 이미지 슬라이드 배너
  - 일시 정지 및 재생 가능
- 일부 섹션별 특정 스크롤 시점이 넘어갈 때 나타나는 fade-in 등의 효과 부여
  - hover 효과 포함
- Product 섹션의 돋보기 버튼 클릭시 modal이 나타나도록 함
  - 다운로드 버튼 클릭 시엔 미리 업로드되어 있는 파일 다운로드(다운로드 되는 파일은 임의의 디자인 시안 파일입니다)
- 동영상 팝업창
